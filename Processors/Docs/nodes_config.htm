General node configuration.

Name:

The default or custom name of the node.

Node ID:

The unique identifier of the node.

Note that you can click the "Node ID" label to copy ID to clipboard.

Description:

Custom discription of the node. This is used defined and has no default.

Job processing slots:

Defines how many slots the node will occupy. E.g. one might want to set encoder nodes to occupy more slots than lighter nodes in order to prevent overload.

Hosts group:

Sets what hosts group the node can run on. This will make sure that the node only runs on the hosts defined in the group. Defining groups is done in the man general FFAStrans configuration.

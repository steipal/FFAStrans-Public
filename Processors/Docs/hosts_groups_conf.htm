This section defines the hosts groups. A host group is a named group of hosts that can be assigned to any workflows or nodes.

Delete group

Deletes the selected group from the list.

Add hosts

Add the host to the group.

Remove

Removes selected host(s) from the group.

New group

Creates a new named group.

Group id

The technical identification of the group.

Note: A maximum of 255 groups can be defined.

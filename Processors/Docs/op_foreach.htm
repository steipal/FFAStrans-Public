This node enables dynamic branching of values in an array.

Array string

Input field for the string represented as an array or a a line feed separated string.

Please check out https://www.w3schools.com/js/js_json_arrays.asp for how to construct a proper array string.

Note that valid arrays can also be constructed by using the $jsonput() funtion in the "Populate" node.

Save value from array index to

Sets the variable on which to save the value from each array index that will be used in the dynamically created branch.

Save index from array to

Sets the variable on which to save the index of each iteration in the array or enter separated string.
